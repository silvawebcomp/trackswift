# TrackSwift

A responsive logistics and package-tracking landing page with a customer tracking experience and an interactive administrator status-update preview.

## Run locally

```powershell
node server.mjs
```

Then open `http://127.0.0.1:4173`.

## Features

- Shipment tracking form and milestone timeline
- Delivery metadata and carrier notes
- Active-shipment administration table
- Status, location, and notification update workflow
- Responsive Tailwind CSS layout
